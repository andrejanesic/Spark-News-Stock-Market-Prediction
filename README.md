# Spark Daily News & Stock Market Prediction

Read headlines and profit from the stock market?

Could it be possible?

In this set of Jupyter Notebooks, I demonstrate how data science and machine learning can be applied to the fields of NLP and stock analysisâ€”with the hopes of gaining insights into how [DJIA](https://en.wikipedia.org/wiki/Dow_Jones_Industrial_Average) stocks perform over time.

The project consists of 4 different notebooks that use [Apache Spark](https://spark.apache.org/) to parse, transform and process data in a cluster of servers.
